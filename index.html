Waifu2X

La mejor herramienta para mejorar la resolución de archivos multimedia

Waifu2x es un algoritmo que mejora la escala de las imágenes a la vez que reduce el ruido. Recibe su nombre del arte de estilo anime conocido como "waifu", con el que se entrenó principalmente. Aunque las waifus constituyeron la mayor parte de los datos de entrenamiento, esta API de waifu2x sigue funcionando bien con fotografías y otros tipos de imágenes. Puedes usar Waifu2x para duplicar el tamaño de tus imágenes a la vez que reduce el ruido.
